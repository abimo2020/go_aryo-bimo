(7) Time Complexity & Space Complexity
==
Resume :
- Time complexity adalah waktu kompleksitas yang dibutuhkan oleh program untuk dikompilasi, dimana kompleksitas tersebut dilihat dari banyaknya operasi yang dilakukan
- Space complexity adalah ruang memory yang dibutuhkan oleh suatu program agar dapat dijalankan, dimana kompleksitas tersebut dipengaruhi oleh jumlah variabel yang dideklarasikan
- Time dan space complexity dilambangkan dengan Big O notation atau O(n)
